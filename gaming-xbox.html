Xbox — Core Safety Checklist

Main risks: voice chat, friend requests, messages, cross-platform play.

Immediate Protection

  • ☐ Set account privacy to Friends Only
  • ☐ Communications (messages/voice) → Friends Only
  • ☐ Friend requests → approval-based
  • ☐ Disable voice chat for younger children (or restrict to friends)
  • ☐ Limit who can see profile info

Ongoing Rules

  • ☐ Monthly friend list review
  • ☐ No adding strangers from lobbies
  • ☐ If a game has chat: prefer private party with known friends
