Discord

Servers + DMs + voice rooms — lock it down before your child uses it.

Social Apps › Discord

Why Discord is high exposure

Discord is built for messaging, group servers, and voice chat. A child can be invited into large public servers, receive DMs from strangers, and be pressured to move into private chats or calls.

Grooming: warning signs Predators & off-platform luring

Quick win (2 minutes)

Goal: stop stranger contact.

1) Block DMs from server members (default risk).

2) Filter message requests / spam.

3) Friends = real-life friends only.

Step-by-step locks (recommended)

A) Privacy & Safety

1) Open Discord → tap your profile (bottom right)
2) Tap Settings (gear icon)
3) Go to Privacy & Safety
4) Turn ON the strongest filtering options available (message request / explicit content / spam filtering)
5) Turn OFF anything that allows strangers to contact your child directly

B) Direct Messages from servers

1) Open the server → tap the server name
2) Go to Privacy Settings
3) Set Direct Messages = OFF (for that server)
4) Repeat for every server your child joins

C) Friend Requests

1) Settings → Friend Requests
2) Allow only the safest option available (ideally: friends-of-friends OFF, server members OFF)
3) Keep the friends list limited to real-life connections

D) Voice / Calls

1) Rule: no private calls with strangers
2) If voice is used, keep it inside known friend groups only
3) If a child is pressured into voice, secrecy, or “don’t tell your parents” — that’s a red flag

House rule (simple + effective)

No secret servers. No secret DMs. Parents can check settings and server list anytime. If anything feels off, the child will not be punished for telling the truth.
