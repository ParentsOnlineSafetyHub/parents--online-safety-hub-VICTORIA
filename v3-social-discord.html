THE GOLDEN RULE

If anyone asks your child to move chats off a game/app (Discord / Snapchat / Telegram / WhatsApp / Instagram DMs) — that’s a RED FLAG.

Your child should tell you immediately. No punishment. No blame. Just safety.

Why Discord is high exposure

Discord is designed for large communities. Kids can be pulled into servers, DMs, and voice calls quickly — often with older users present. Predators commonly use Discord as the “next step” after meeting kids in games.

  • Servers: public invite links, massive communities, mixed ages.
  • DMs: strangers can message if settings allow it.
  • Voice: fast isolation and escalation (private calls/channels).
  • Private servers: hidden communities and “secret” spaces.

Quick win (60 seconds)

If you do nothing else today: Turn OFF DMs from server members, tighten friend requests, and avoid public servers for kids.

Step-by-step safety setup (recommended)

1) Lock who can message your child

User Settings → Privacy & Safety
• Turn ON: Keep me safe (highest filter)
• Turn ON: Filter direct messages
• Review: Direct Messages settings (keep tight)

2) Turn OFF DMs from server members (critical)

For every server your child joins:
Open the Server → Server Settings / Privacy Settings
• Turn OFF: Direct Messages from server members
• Turn OFF: Friend Requests from server members if available

3) Restrict friend requests

User Settings → Friend Requests
• Prefer: Friends of Friends only (or approved contacts only)
• Avoid: allowing requests from everyone

4) Voice/Calls: lock it down

• Avoid voice chats with unknown people
• If voice is used: friends-only, and parent is nearby
• No “private calls” with strangers — ever

Parent rule: Kids do not join random public servers. Servers must be parent-approved.

Server rules for kids (simple, non-negotiable)

1) No public servers without parent approval.

2) DMs OFF from server members (every server).

3) No private voice calls with anyone you don’t know in real life.

4) Don’t share: name, school, suburb, photos, location, or schedules.

5) If someone asks for secrets or moves to another app → tell parent immediately.

Kid-safe script: “If you ever feel weird about a chat, you won’t be in trouble. Screenshot it and tell me straight away.”

Red flags (Discord-specific)

  • “Don’t tell your parents.” / “They won’t understand.”
  • Private server invites “just for you.”
  • Requests for voice calls late at night.
  • Gifts, Nitro offers, “I’ll help you if…” trades.
  • Sexual jokes, porn links, grooming language, “education.”
  • Any push to Snapchat/Telegram/WhatsApp/Instagram DMs.
Red Flags Reporting

What to do if something happens

1) Screenshot/save messages and usernames.
2) Block the user and leave the server.
3) Report on Discord + report through POSH Reporting guidance.
4) Reassure your child: they did the right thing telling you.
