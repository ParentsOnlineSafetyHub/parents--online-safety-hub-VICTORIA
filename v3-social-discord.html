Social Apps › Discord

Quick win (60 seconds)

If you do nothing else today: block DMs from server members and disable friend requests from strangers.

Step-By-Step (simple clicks)

On the child’s Discord account:

1) Open Discord → tap your profile icon
2) Go to Settings (⚙️) → Privacy & Safety
3) Turn OFF: Allow direct messages from server members
4) Turn ON / highest: Message Requests filtering (if shown)
5) Go to Friend Requests
6) Turn OFF: Everyone (only allow friends-of-friends if needed)
7) Under Safe Direct Messaging choose the strictest option (scan/filter)
8) Review joined servers → leave any server not created/approved by a parent

Parent rule: No “random servers”. Servers must be approved and limited.

Boundary rule (keep it simple)

No DMs with strangers. No private chats outside approved friends. If someone asks to move chat elsewhere (Snap/Telegram/etc) = block + tell a parent.
