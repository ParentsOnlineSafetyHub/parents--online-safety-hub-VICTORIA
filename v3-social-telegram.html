Why Telegram needs attention

Telegram supports groups, channels, direct messages, and bots. Strangers can attempt contact if settings are open.

Step-by-step settings

1) Privacy & Security
Telegram → Settings → Privacy and Security.

2) Phone Number
Set “Who can see my phone number” to Nobody.
Set “Who can find me by number” to My Contacts.

3) Calls & Groups
Set Calls to My Contacts.
Set Groups/Invites to My Contacts.

4) Forwarded messages / profile photos
Restrict who can see profile photos and who can add them.

5) 2-Step Verification
Enable 2-step verification and keep the code private.

Rule

If a message comes from someone unknown: don’t reply → screenshot → block → report.
