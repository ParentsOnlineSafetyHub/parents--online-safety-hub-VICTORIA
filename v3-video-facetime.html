FaceTime Safety Guide

Contacts-only. Use Screen Time to control access.

Why FaceTime Needs Supervision

FaceTime is often viewed as “family safe”, but it still allows direct video contact. If a child can be reached by unknown Apple IDs/emails, risk increases.

Step-by-Step Safety Setup

1) Use Screen Time to restrict FaceTime

Settings → Screen Time → Content & Privacy Restrictions → Allowed Apps → control FaceTime access.

2) Enforce contacts-only

Rule: Only accept calls from real-life known contacts. Unknown caller = decline + block.

3) Keep calls in shared spaces (younger kids)

No bedroom video calls. Shared spaces reduce risk automatically.

4) Apple ID security

Keep Apple ID password private. Use Family Sharing if possible. Avoid sharing Apple login details widely.

House Rule

No video calls with strangers. No “secret calls”. No exceptions.
