Microsoft Teams Safety Guide

School platform — still requires control.

Why Teams Needs Attention

Often used for school. Includes chat, private calls, file sharing. Children can message classmates privately.

Step-by-Step Safety Setup

1. Profile → Settings → Privacy

• Review contact permissions • Limit who can message

2. Meetings

• Disable anonymous join • Restrict screen sharing

3. Chat

• Monitor private chats • Encourage school accounts only

4. Parent Rule

If Teams is used for school — use school login only, not personal accounts.
