PlayStation — Core Safety Checklist

Main risks: party chat, messages, friend requests, adult lobbies.

Immediate Protection

  • ☐ Privacy settings tightened (profile visibility limited)
  • ☐ Messages → Friends Only
  • ☐ Party chat → Friends Only / restricted
  • ☐ Friend requests → restrict as needed
  • ☐ Purchases → require passcode

Ongoing Rules

  • ☐ Monthly friend list review
  • ☐ No private chats with strangers
  • ☐ Use private parties where possible
