Primary Risk Areas

Immediate Lockdown Settings

  • ☐ Restricted Mode ON (passcode protected)
  • ☐ Account set to Private
  • ☐ DMs restricted
  • ☐ Comments limited
  • ☐ Disable LIVE where possible
  • ☐ Screen-time boundary set

House Rule

No DMs with unknown accounts. If someone asks for secrecy or moves the chat — red flag.
