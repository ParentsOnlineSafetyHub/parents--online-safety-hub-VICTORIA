Snapchat Safety Guide

Snapchat is high exposure. Private messaging + disappearing content = increased risk.

Why Snapchat Needs Strict Settings

Snapchat allows: • Private messaging • Disappearing messages • Live location (Snap Map) • Quick Add from strangers • Group chats • Video calls Default settings are NOT child-safe.

Step-by-Step Safety Setup

1. Profile → Settings (⚙ top right)

2. Contact Me

Set to: My Friends Not “Everyone”.

3. View My Story

Set to: Friends Only Or use Custom and restrict further.

4. See Me in Quick Add

Turn this OFF. This stops strangers adding your child easily.

5. Snap Map (Location)

Go to Snap Map → Settings → Enable Ghost Mode. Never allow live location for children.

6. Privacy Controls → Family Center

If available, enable Family Center to monitor friend list and communication patterns.

7. Blocking & Reporting

Hold down on username → Manage Friendship → Block / Report.

Non-Negotiable Parent Rules

• No strangers added • No sharing school name publicly • No public stories • No location sharing • No private video calls with unknown contacts
