Webex Safety Guide

Meeting links are access keys. Lock entry, lock sharing, lock chat.

Why Webex Needs Supervision

Webex can be used for school, tutoring, clubs, and family calls. The main risks are meeting links being shared, unknown participants joining, screen sharing, and chat/file sharing.

Step-by-Step Safety Setup

1) Don’t share links publicly

Treat meeting links like a house key. Only send to trusted adults. Never post in group chats, gaming chats, or public comments.

2) Use meeting passwords / lobby controls

When creating a meeting: • Turn on password (if available) • Use lobby/waiting room (host approval before entry)

3) Restrict screen sharing

In meeting options or host controls: • Screen share = Host only (recommended)

4) Control chat and file sharing

If options exist: • Disable private chat • Disable file transfer

5) Lock the meeting once everyone is in

If Webex provides “Lock meeting” as host — use it.

House Rule

If an unknown person appears in a call: end the meeting immediately, then block/report.
