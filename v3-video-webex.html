Why Webex needs supervision

Webex is common for workplaces and some schools. Risks come from meeting links, unknown participants, screen share, and chat.

Step-by-step: safer Webex meetings

1) Don’t share links publicly
Send links only to trusted adults/approved groups.

2) Use meeting passwords / lobby controls
Enable meeting password and require host approval where possible.

3) Disable participant screen sharing
Set sharing to host only for children’s meetings.

4) Control chat
Disable private chat and file sharing where possible.

5) Supervise younger kids
Keep calls in common areas and end meetings if strangers appear.
