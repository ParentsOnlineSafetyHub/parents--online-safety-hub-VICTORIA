Nintendo Safety Guide

Use Nintendo Parental Controls. Lock chat + spending + time.

What matters on Nintendo

Nintendo Switch controls can restrict: online communication, game ratings, spending, and play time.

Step-by-step (quick setup)

1) Install Nintendo Switch Parental Controls app

Set up supervision from your phone (recommended).

2) Set age rating

Choose an age restriction profile or custom settings.

3) Restrict communication

Disable or limit communication features where possible.

4) Spending controls

Require password for purchases and limit store access.

5) Play time limits

Set daily limits + bedtime restrictions.
