Zoom Safety Guide

Meeting links must be controlled.

Why Zoom Needs Supervision

Zoom allows meeting links, screen sharing, chat, and recording. If links are shared publicly, strangers can enter.

Step-by-Step Safety Setup

1. Settings → Security

• Enable Waiting Room • Require passcode for meetings

2. In-Meeting Settings

• Disable participant screen sharing • Disable file transfer • Disable private chat

3. Recording

• Disable automatic recording • Monitor saved recordings

4. Teach This Rule

Never post meeting links in public chats or gaming servers.
