Platform Setup (Quick Steps)

  • ☐ Settings → Users & Accounts → Privacy
  • ☐ Set messages to Friends Only
  • ☐ Restrict voice chat / party chat
  • ☐ Family & Parental Controls → spending limits / approvals
  • ☐ Review friends list monthly

If menus differ, use the Settings search and type: privacy, family, messages.
