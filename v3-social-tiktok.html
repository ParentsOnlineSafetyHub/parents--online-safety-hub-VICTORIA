Priority risks

TikTok includes public comments, DMs, lives, stitching/duets, and rapid content discovery. Keep accounts private and reduce contact from strangers.

Step-by-step settings

1) Make the account private
TikTok → Profile → ☰ → Settings and privacy → Privacy → Private account ON.

2) Direct messages
Privacy → Direct messages → set to No one (best for kids) or Friends.

3) Comments
Privacy → Comments → set who can comment (Friends) + filter keywords.

4) Duet / Stitch / Downloads
Privacy → Duet/Stitch → Friends or Off.
Privacy → Downloads → Off.

5) Screen Time / Restricted Mode
Digital Wellbeing → Screen Time Management + Restricted Mode ON (set a passcode you control).

Rule

No DMs. Kids can watch, but strangers cannot contact them.
