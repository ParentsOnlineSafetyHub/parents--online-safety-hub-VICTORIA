Gaming › Fortnite

Quick win (60 seconds)

Biggest risk is voice chat and random parties. Set chat to Friends only or Off.

Step-By-Step (simple clicks)

1) Open Fortnite → go to Settings (gear icon)
2) Go to Audio / Voice Chat settings
3) Set Voice Chat = Off (or Friends only)
4) Set Voice Chat Method = Push-to-Talk (if enabled)
5) Go to Privacy / Account & Privacy
6) Set Party Privacy = Private (invite-only)
7) Review Friend Requests — only accept known friends

If your child uses console/PC, platform chat controls are the “second lock”: Platforms hub.
