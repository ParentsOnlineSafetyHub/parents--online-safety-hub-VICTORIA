Why Steam needs scrutiny

Steam includes friends, chat, group chat, community profiles, and voice/text chat in games. If privacy is open, strangers can message and add your child.

Step-by-step: lock Steam down

1) Make the profile private
Steam → Profile → Edit Profile → Privacy Settings.
Set:
• My profile: Private
• Friends list: Private
• Game details: Private (or Friends Only)

2) Control friend requests
Teach a simple rule: No accepting friend requests unless the person is known offline.

3) Disable or restrict chat
In each game, look for chat/voice settings and set to:
• Friends only, or
• Off (best for younger kids)

4) Family View (PIN lock)
Steam → Settings → Family → Family View → Enable.
Use a PIN so store/community settings can’t be changed without you.

5) Keep purchases locked
Use Family View and don’t store payment details on the child’s account.

Quick rule

No private messaging with strangers. Any odd message → screenshot → block → report.
