Gaming › Roblox

Quick win (60 seconds)

If you do nothing else today: lock communication, lock who can add friends, and add a parent PIN.

Step-By-Step (simple clicks)

On the child’s Roblox account:

1) Open Roblox → tap More (⋯) / menu
2) Go to Settings → Privacy
3) Set Who can message me = No one (or Friends only)
4) Set Who can chat with me in-app = No one (or Friends only)
5) Set Who can join me in experiences = Friends (not Everyone)
6) Set Who can follow me into experiences = No one / Friends
7) Set Who can add me as a friend = Friends (avoid Everyone)
8) Turn off Camera / Microphone access at device level unless needed

Parent controls (recommended):

1) Go to Settings → Parental Controls
2) Turn on Parental Controls
3) Set a Parent PIN
4) Review Spending / Purchases and lock it down

If your child plays on console/PC, do the platform rules too: Platforms hub.

Boundary rule (keep it simple)

No private chats with strangers. Friends list must be people you know in real life. If a “friend” is unknown, remove and report.
