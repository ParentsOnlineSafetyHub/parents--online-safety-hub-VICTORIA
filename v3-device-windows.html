Device Controls › Windows

Quick win (60 seconds)

Create a separate child Windows account (not admin). Most safety issues come from kids using an admin account.

Step-by-step (simple clicks)

1) On Windows: Settings → Accounts → Family
2) Add a child account (Microsoft account) and link it to your family group
3) Make sure the child account is Standard, not Administrator

Now set the core protections:
4) Turn on Screen time limits (daily + bedtime)
5) Turn on Content filters (web + apps + age limits)
6) Require approval for purchases
7) Review installed apps/games and remove anything risky

Recommended boundary

No admin access. No installs without a parent. If a game needs installing, parent does it and checks in-game chat settings immediately.
