VR Safety Guide

VR is immersive. If it has voice chat, it needs scrutiny.

What matters in VR

VR platforms often include open voice chat, public rooms, and adult users. This increases risk for kids fast if settings are left open.

Step-by-step (quick setup)

1) Use child-appropriate account settings

Set correct age and enable parental controls if available.

2) Restrict voice chat

Set voice chat to Friends only or Off for kids.

3) Restrict public rooms

Avoid open public lobbies. Prefer private sessions with known friends.

4) Time limits + supervision

VR should be supervised for younger children. Set time windows to prevent overuse.
