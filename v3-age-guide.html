Age Guide (Simple Rules That Work)

One standard. Same boundaries. Less arguing, more safety.

How to use this page

Pick your child’s age range. Apply the rules. Then lock the device/platform settings to match. If rules don’t match settings, kids will always find a gap.

Non-negotiables (all ages)

1) No private chats with strangers.
2) No bedroom voice/video calls for younger kids.
3) Location sharing OFF.
4) Parents keep passwords + approve installs.
5) If an adult asks for secrecy → tell a parent immediately.
6) If it has chat/DMs → it gets locked down first.

Ages 0–6 (foundation years)

Recommended: no social apps, no online chat, no open multiplayer.
Best tools: YouTube Kids (restricted), educational apps, supervised screen time.

Rules:
• Screens only in shared areas
• No headphones (parents need to hear content)
• Content is parent-selected only
• Short sessions, regular breaks

Ages 7–9 (curiosity years)

Recommended: no social media, no DMs, no open voice chat.
Games should be “friends only” and supervised.

Rules:
• Friends list = real life friends only
• Multiplayer only with friends (not public lobbies)
• No voice chat (or friends-only at most)
• Parent check-ins: weekly

Ages 10–12 (early independence)

This is the danger zone for grooming — kids want independence but don’t understand manipulation yet.

Recommended:
• No public profiles
• No random friend adds
• DMs locked to friends only
• Voice chat = off or friends-only

Rules:
• Parents review privacy settings monthly
• Parents can see friend list (no secrets)
• No disappearing messages apps (or heavily locked down)
• All devices charge overnight in a shared area

Ages 13–15 (high exposure years)

Teens will push boundaries. You don’t “ban everything” — you set structure, supervision, and consequences.

Recommended:
• Private account by default
• DMs: friends only
• No meeting strangers
• No sharing school, suburb, uniform, or daily routine

Rules:
• 1–2 social apps only (not 6 at once)
• Weekly quick check (5 mins): privacy + friends + messages requests
• If they break rules: restrictions increase (not yelling, just consequences)

Ages 16–17 (training for adulthood)

This is where you teach judgment, not control.

Recommended:
• They manage most settings — but parents still guide
• Teach reporting, blocking, and safe boundaries
• Talk about consent, pressure, and manipulation

Rules:
• No sharing intimate content (ever) — it can be used as leverage
• No meeting online contacts alone
• If something feels “off” → talk early

Common parent mistakes (quick fixes)

Mistake: “My kid wouldn’t do that.”
Fix: Lock settings anyway — prevention is not punishment.

Mistake: “I’ll just take the phone.”
Fix: Use calm consequences + settings + education.

Mistake: “They’ll hate me.”
Fix: They’ll respect you later. Safety isn’t popularity.
