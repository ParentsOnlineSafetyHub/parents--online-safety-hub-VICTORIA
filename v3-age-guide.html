POSH Aussie

Age Guide

Age-appropriate boundaries reduce risk more than any single app setting.

The POSH baseline (for every age)

1) Device controls ON (installs, screen time, filters).

2) No secret accounts. Parents can check anytime.

3) If it has chat/DMs/calls/groups — it must be locked down.

4) Devices charge overnight in a shared area.

Lock Devices (Step 2) House Rules (Step 3)

Under 10

Goal: supervised learning + play only.

✅ No public chat / no DMs / no voice chat with strangers.

✅ Friends = real-life friends only (approved by parent).

✅ “Kid modes” only (YouTube Kids / family profiles / content filters).

✅ Co-play: parent nearby for online games.

10–12

Goal: build habits + boundaries.

✅ Private accounts by default.

✅ DMs friends-only (or OFF if possible).

✅ No group chats with unknown people.

✅ Weekly 5-minute settings check together.

13–15

Goal: guided independence.

✅ Teach red flags: flattery, secrecy, gifts, moving to private chat, “don’t tell your parents”.

✅ Location sharing OFF.

✅ Friend list review weekly (calm, not accusing).

✅ Platforms locked first → then games/apps.

16+

Goal: trust + accountability.

✅ Keep non-negotiables: no secret accounts, no private meets, no sharing location.

✅ Teach: screenshot evidence, block/report fast, ask for help early.

✅ Freedom increases with honesty + consistency.

Simple parent script (use this)

“I’m not here to control you. I’m here to protect you. If something ever feels off — you won’t be in trouble. We’ll handle it together.”
