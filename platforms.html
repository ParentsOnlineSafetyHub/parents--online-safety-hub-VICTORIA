Platform Safety Checklists

Most parents don’t need “more information” — they need steps they can follow. Use the buttons below to jump straight to the platform your child uses most.

Roblox Fortnite YouTube Snapchat Instagram TikTok Discord

Tip: Do the top 5 steps first. If you only have 10 minutes, settings that control who can contact your child are the priority.

Roblox — Safety Checklist

Main risks: chat, private servers, friend requests, inappropriate experiences.

Do This First (Fast Protection)

  1. Turn on Parental Controls and set a parent PIN.
  2. Set Communication to the strictest level you’re comfortable with (ideally “Friends only” or “Off” for younger kids).
  3. Restrict who can message / chat / party.
  4. Limit access to private servers (who can join / who they can join).
  5. Set content maturity to age-appropriate.

Privacy & Contact Controls

  • ☐ Who can message? → Friends Only (or Off)
  • ☐ Who can chat in experiences? → Friends Only (or Off)
  • ☐ Who can join private servers? → Connections Only
  • ☐ Who can follow into experiences? → Restrict
  • ☐ Disable or restrict party chat where possible

Spending & Time

  • ☐ Set spending controls / require approval
  • ☐ Set screen-time expectations (routine, not punishment)

Fortnite — Safety Checklist

Main risks: voice chat, friends-of-friends, random squad interactions, DMs via Epic.

Do This First (Fast Protection)

  1. Enable Epic Games Parental Controls and set a 6-digit PIN.
  2. Set Voice Chat to Friends Only or Nobody for younger kids.
  3. Set Text Chat to Friends Only or Nobody.
  4. Block purchases or require PIN for spending.
  5. Review friend list monthly.

Recommended Settings

  • ☐ Voice Chat → Friends Only / Nobody (depending on age)
  • ☐ Text Chat → Friends Only / Nobody
  • ☐ Add Friends → restrict / approval-based where possible
  • ☐ Purchases → require PIN
  • ☐ Keep account email controlled by parent

If your child is very young: consider disabling voice chat entirely. Most grooming starts with “friendly” voice conversations.

YouTube — Safety Checklist

Main risks: algorithm recommendations, comments, DMs via external links, mature content.

Do This First (Fast Protection)

  1. If under 13: use YouTube Kids or a supervised experience.
  2. Turn on Restricted Mode (Family Link / YouTube settings).
  3. Disable Autoplay.
  4. Pause or review watch/search history if needed.
  5. Teach “No clicking links in descriptions/comments”.

Recommended Settings

  • ☐ Restricted Mode ON
  • ☐ Autoplay OFF
  • ☐ Limit Shorts bingeing (time boundaries)
  • ☐ Teach: “Comments aren’t a safe place”

Snapchat — Safety Checklist

Main risks: Snap Map location, quick-add strangers, disappearing messages, private photos.

Do This First (Fast Protection)

  1. Turn on Ghost Mode in Snap Map (location OFF).
  2. Set who can contact your child → Friends Only.
  3. Disable or restrict Quick Add.
  4. Review privacy settings together monthly.
  5. Discuss: “No photos to online-only contacts.”

Recommended Settings

  • ☐ Snap Map → Ghost Mode ON
  • ☐ Contact Me → Friends Only
  • ☐ View My Story → Friends Only (or Custom)
  • ☐ Quick Add → OFF / restricted

Instagram — Safety Checklist

Main risks: DMs, follower requests, explore content, older accounts contacting teens.

Do This First (Fast Protection)

  1. Set the account to Private.
  2. Set message requests to the strictest level (limit unknowns).
  3. Enable Supervision (Family Center) if appropriate.
  4. Limit comments and mentions to people they know.
  5. Review followers list monthly.

Recommended Settings

  • ☐ Account → Private
  • ☐ Message requests → limit unknowns
  • ☐ Tags/Mentions → People You Follow
  • ☐ Sensitive content controls → tighter
  • ☐ Supervision (if you choose) → set up via Family Center

TikTok — Safety Checklist

Main risks: addictive feed, DMs, mature content, LIVE features.

Do This First (Fast Protection)

  1. Turn on Restricted Mode.
  2. Set account to Private (if applicable).
  3. Use Family Pairing for parent-managed controls (optional but powerful).
  4. Limit DMs and commenting.
  5. Create clear time boundaries (daily cap).

Recommended Settings

  • ☐ Restricted Mode ON (passcode protected)
  • ☐ Family Pairing ON (if suitable)
  • ☐ DMs → strict / limited
  • ☐ Comments → friends only (where available)
  • ☐ Screen time limits (consistent routine)

Discord — Safety Checklist

Main risks: DMs from strangers, servers, explicit content, “message requests”.

Do This First (Fast Protection)

  1. In Privacy & Safety, enable filtering for explicit media and spam.
  2. Disable DMs from server members unless necessary.
  3. Use Family Center (optional) to keep informed without reading messages.
  4. Teach your child: never accept random server invites.
  5. Review servers joined monthly.

Recommended Settings

  • ☐ Explicit content filter → stronger filtering
  • ☐ DMs from server members → OFF
  • ☐ Message requests → review carefully
  • ☐ Family Center (optional) → set up for visibility
