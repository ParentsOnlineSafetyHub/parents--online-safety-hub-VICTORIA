Platform Safety Guides (Gaming + Social)

Any platform with chat, DMs, voice, or the ability to add strangers needs scrutiny. This page gives you structured checklists for the most common apps and games in Australia.

Priority order: Who can contact your child → Who can see them → What content they can access → Spending → Time boundaries.

Roblox (Highest Priority)

Primary Risks: open chat, friend requests from strangers, private servers, user-created content, in-game currency scams.

Immediate Protection Steps

  • ☐ Enable Parental Controls
  • ☐ Set a Parent PIN (don’t share it)
  • ☐ Communication / Chat → Friends Only (or OFF for younger kids)
  • ☐ Who can message? → Friends Only
  • ☐ Who can join private servers? → Connections Only
  • ☐ Review friend list monthly (remove unknown accounts)

Content + Spending

  • ☐ Age-appropriate content/maturity settings tightened
  • ☐ Spending controls enabled / require approval
  • ☐ Teach: “No moving chats to Discord/Snapchat from Roblox”

Fortnite

Primary Risks: voice chat with strangers, random squads, friends-of-friends contact, cross-platform messaging.

Immediate Protection Steps

  • ☐ Enable Epic Games Parental Controls
  • ☐ Set a PIN
  • ☐ Voice Chat → Friends Only or Nobody (age dependent)
  • ☐ Text Chat → Friends Only or Nobody
  • ☐ Purchases → Require PIN
  • ☐ Friend list review monthly

House Rule

  • ☐ If playing with friends: prefer known friends in private party
  • ☐ No accepting random invites / “carry you” offers

Minecraft

Primary Risks: public servers, chat, voice via external apps, strangers inviting kids to private worlds/servers.

Immediate Protection Steps

  • ☐ Prefer private worlds with known friends/family
  • ☐ If using servers: only approved, moderated servers
  • ☐ Chat → restricted where possible (or supervised)
  • ☐ Microsoft/Xbox privacy settings: limit who can communicate and add friends

House Rule

  • ☐ No joining random servers from YouTube/TikTok links
  • ☐ No moving chats to Discord/Snapchat without parent approval

Call of Duty (COD)

Primary Risks: unfiltered voice chat, adult lobbies, direct messages, harassment, mature themes.

Immediate Protection Steps

  • ☐ Voice chat → mute all / friends only (age dependent)
  • ☐ Disable cross-play communication where possible
  • ☐ Restrict friend requests / messages
  • ☐ Keep lobbies private with known friends when possible

Reality Check

  • ☐ COD lobbies often contain adult language/content — treat as an “adult environment”
  • ☐ If a child plays: tighten chat, supervise, and set clear rules

Grand Theft Auto (GTA / GTA Online)

Primary Risks: adult themes, voice chat with adults, roleplay servers, harassment, exposure risk is high.

Immediate Protection Steps

  • ☐ If underage: strongly consider not allowing GTA Online
  • ☐ If allowed (older teen): voice chat restricted and privacy tightened
  • ☐ Keep sessions private with known friends where possible
  • ☐ Console account restrictions (PlayStation/Xbox): communication + friend requests limited

Non-Negotiables

  • ☐ No roleplay servers unless parent-approved and moderated
  • ☐ No private voice chats with strangers

YouTube

Primary Risks: algorithm exposure, mature content, comments, external links, bingeing.

Immediate Protection Steps

  • ☐ Under 13 → YouTube Kids or supervised experience
  • ☐ Restricted Mode → ON
  • ☐ Autoplay → OFF
  • ☐ Teach: “No clicking links in comments/descriptions”
  • ☐ Time boundary for Shorts

Snapchat

Primary Risks: Snap Map location, Quick Add strangers, disappearing messages/images.

Immediate Protection Steps

  • ☐ Snap Map → Ghost Mode ON
  • ☐ Contact Me → Friends Only
  • ☐ View My Story → Friends Only (or Custom)
  • ☐ Quick Add → OFF / restricted
  • ☐ No photos to online-only contacts

Discord

Primary Risks: DMs from strangers, servers, explicit content, grooming attempts via communities.

Immediate Protection Steps

  • ☐ Disable DMs from server members
  • ☐ Strong explicit media filtering enabled
  • ☐ Message requests restricted
  • ☐ Only approved servers (moderated)
  • ☐ Review servers monthly
Back To Top

Instagram

Primary Risks: DMs, message requests, older accounts contacting teens, explore feed exposure.

Immediate Protection Steps

  • ☐ Account → Private
  • ☐ Message requests → limit unknowns
  • ☐ Tags/Mentions → people they follow
  • ☐ Sensitive content controls → tighter
  • ☐ Review followers monthly
Back To Top

Facebook

Primary Risks: DMs, groups, older accounts contacting minors, marketplace exposure, privacy leakage.

Immediate Protection Steps

  • ☐ Profile privacy tightened (friends only)
  • ☐ Friend requests → only known people
  • ☐ Messaging → restrict unknown message requests
  • ☐ Location + personal details hidden
  • ☐ Review groups joined / pages followed
