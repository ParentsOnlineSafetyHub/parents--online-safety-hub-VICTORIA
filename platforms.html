Platform-Specific Protection Guides

Each platform has different risks and different safety controls. Below are structured checklists for the most commonly used platforms across Australia.

If you only have time for one thing — prioritise settings that control: Who Can Contact Your Child.

Roblox Fortnite YouTube Snapchat Instagram TikTok Discord

Roblox

Primary Risks: open chat, private servers, friend requests from strangers, inappropriate user-created experiences.

Immediate Protection Steps

  • ☐ Turn on Parental Controls
  • ☐ Set and protect a Parent PIN
  • ☐ Communication → Friends Only (or Off for younger children)
  • ☐ Who can message? → Friends Only
  • ☐ Restrict private server access
  • ☐ Review friend list monthly

Advanced Controls

  • ☐ Limit experience maturity level
  • ☐ Disable “Follow Me Into Experiences”
  • ☐ Restrict spending / require approval

Fortnite

Primary Risks: voice chat with strangers, random squads, cross-platform messaging.

Immediate Protection Steps

  • ☐ Enable Epic Parental Controls
  • ☐ Set a 6-digit PIN
  • ☐ Voice Chat → Friends Only or Nobody
  • ☐ Text Chat → Friends Only or Nobody
  • ☐ Require PIN for purchases

Ongoing Protection

  • ☐ Review friend list monthly
  • ☐ Discuss not accepting random squad invites
  • ☐ Disable voice chat entirely for younger children
