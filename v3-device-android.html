Device Controls › Android

Quick win (60 seconds)

Install Google Family Link on the parent phone and link the child account. This gives you app approvals, time limits, and content filters.

Step-by-step (simple clicks)

1) On the parent phone, install Family Link
2) Open it → create/link the child’s Google account
3) On the child phone, follow prompts to connect the device

Now set the core protections:
4) Turn ON App approvals (child can request, parent approves/denies)
5) Set Daily limits and a bedtime
6) Set Content restrictions (Play Store age limits + SafeSearch)
7) Review installed apps and remove anything risky
8) Lock down permissions (camera/mic/location) where not needed

Recommended boundary

No installs without approval. If an app is needed, approve it, then immediately open and configure privacy settings.
