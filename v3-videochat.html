Video & Live Communication

Live contact needs higher supervision. Lock entry, lock sharing, keep calls safe.

Why video platforms require stricter rules

Video/voice apps are not “bad” — they’re powerful. The risks increase when kids can:

• Join calls via forwarded links
• Receive calls from unknown contacts
• Screen share privately
• Be pulled into 1-on-1 conversations
• Be recorded or asked to record

POSH focuses on prevention: calm rules, safer defaults, consistent supervision.

Universal safety settings (apply everywhere)

1) Never post meeting links publicly. Links are access keys.

2) Enable approval entry. Waiting room / lobby / host approval.

3) Restrict screen sharing. Host only (recommended for kids).

4) Restrict chat and file sharing. Disable private chat if possible.

5) No bedroom calls for younger kids. Shared spaces reduce risk automatically.

6) Unknown person joins = end call immediately.
