Apple Screen Time

iPhone & iPad safety. Use Apple Screen Time correctly to control installs, communication, and device usage.

Quick win (60 seconds)

Turn on Screen Time and set a Screen Time Passcode (parent-only). This prevents children from changing restrictions.

Step-by-step setup

1) Open Settings → Screen Time
2) Tap Turn On Screen Time
3) Tap Use Screen Time Passcode → set a parent passcode

Core protections:
4) Turn on Content & Privacy Restrictions
5) iTunes & App Store Purchases → restrict installs/deletes
6) App Limits → limit social and gaming apps
7) Downtime → set bedtime hours
8) Communication Limits → contacts only
9) Always Allowed → only essential apps

Recommended boundary

No new apps without parent approval. If a child needs an app for school or communication, the parent installs it first and checks privacy settings.
