Device Controls › Apple

Quick win (60 seconds)

Turn on Screen Time and set a Screen Time Passcode (parent-only). This stops kids changing limits.

Step-by-step (simple clicks)

1) Open Settings → Screen Time
2) Tap Turn On Screen Time
3) Tap Use Screen Time Passcode → set a parent passcode (don’t share it)

Now set the core protections:
4) Content & Privacy Restrictions → turn ON
5) iTunes & App Store Purchases → “Don’t Allow” installs/deletes (or require password)
6) App Limits → add limits for Social/Games (start with 30–60 mins)
7) Downtime → set bedtime lockout hours
8) Communication Limits → limit to Contacts only (or specific contacts)
9) Always Allowed → only allow essentials (calls to parents, school apps, etc.)

Recommended boundary

No new apps without parent approval. If the child needs an app for school, parent installs it and configures privacy first.
