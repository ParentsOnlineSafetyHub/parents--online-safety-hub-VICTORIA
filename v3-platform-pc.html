PC Safety Guide

PC is powerful. That means stronger supervision is required.

What matters on PC

PC allows open installs, open chat platforms, and access to adult content if not restricted. Start with device controls, then lock games and apps.

Step-by-step (quick setup)

1) Create a separate child login

Do not share admin accounts. Child account should not install apps without permission.

2) Enable Microsoft Family Safety (Windows)

Set screen time, web filters, app limits, and purchase controls.

3) Browser safety

Turn on SafeSearch, restrict adult sites, and lock browser permissions.

4) App installs

Block unknown installers. Only allow approved games/apps.
