Skype Safety Guide

Lock down direct contact. Default settings are not child-safe.

Why Skype Needs Attention

Skype allows direct messaging, video calls, screen sharing, and file transfers. If a child can receive contact from anyone, risk increases.

Step-by-Step Safety Setup

1. Open Skype → Profile → Settings

2. Privacy → Contacts

• Set “Allow calls from” → Contacts only • Disable “Automatically answer calls”

3. Messaging

• Set messages to Contacts only • Disable read receipts if needed

4. Screen Sharing

• Supervise during screen sharing • Teach child never to share screen with strangers

5. Block Unknown Contacts

• Remove unknown contacts • Use Block + Report where necessary
