Primary Risk Areas

  • ☐ Friend requests and chat from strangers
  • ☐ Community forums and profile comments
  • ☐ Links and scams (trades, “free skins”, giveaways)
  • ☐ Adult content in some community areas

Immediate Lockdown Settings

  • ☐ Profile privacy → Friends Only (or Private)
  • ☐ Comments on profile → Friends Only
  • ☐ Inventory visibility → Friends Only
  • ☐ Restrict who can add them as a friend
  • ☐ Teach: “No trades / links / logins from DMs”

House Rule

No clicking links or logging in via links from messages. If it feels rushed or secretive — it’s a red flag.
