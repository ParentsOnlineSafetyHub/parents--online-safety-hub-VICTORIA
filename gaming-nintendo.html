Nintendo — Core Safety Checklist

Main risks: friend codes, online play, voice chat via external apps, purchases.

Immediate Protection

  • ☐ Use Nintendo Parental Controls app
  • ☐ Restrict online communication where possible
  • ☐ Manage friend interactions (approved only)
  • ☐ Purchases restricted / require approval
  • ☐ Screen-time limits set

House Rule

  • ☐ No sharing friend codes publicly
  • ☐ Voice chat rules still apply (even if via phone apps)
