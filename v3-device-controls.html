Device Safety

Your strongest safety layer. Lock the device first, then the platform, then the app.

Start here first

Most parents jump straight into apps and games.

Device safety comes first because it controls installs, screen time, filters, app permissions, and account changes.

If the device is open, everything else is weaker

What to lock first on every device

1) Restrict app installs

2) Set screen time / downtime

3) Lock explicit content

4) Turn off unknown camera / microphone access

5) Use a parent-controlled passcode / PIN

Apple (iPhone / iPad)

Go to Settings → Screen Time
Turn on Content & Privacy Restrictions
Lock app installs, in-app purchases, explicit content, and web filtering

Best Apple setup: Screen Time passcode + app install restrictions + content limits.

Android

Use Google Family Link
Restrict installs, manage app approvals, set time limits, and filter content

Best Android setup: Family Link + Play Store approval + app-level review.

Windows / PC

Use Microsoft Family Safety
Add the child account, restrict websites, filter games, and manage time

Best PC setup: child account + web filters + game age restrictions.
