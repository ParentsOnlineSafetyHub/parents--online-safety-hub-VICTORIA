Device Controls

Your strongest protection layer. Lock the device first.

Quick start (3 wins in 5 minutes)

1) Turn on parental controls / family controls.

2) Block new app installs without approval.

3) Set screen-time limits + bedtime charging outside bedrooms.

Next: Platforms Then: Social Apps Then: Gaming

Non-negotiables (recommended)

These settings reduce risk across every app and game:

✅ App installs require parent approval

✅ Explicit content filters ON

✅ Web filtering ON (age-appropriate)

✅ Location sharing OFF unless required

✅ Device charging overnight in a shared area

Back: Age Guide House Rules

Apple (iPhone / iPad)

Settings → Screen Time
1) Turn on Screen Time
2) Content & Privacy Restrictions → ON
3) iTunes & App Store Purchases → Installing Apps = Don’t Allow (or Require Password)
4) Content Restrictions → set age-appropriate limits
5) Downtime → set bedtime hours

Android (Google Family Link)

Install: Google Family Link
1) Add your child’s Google account
2) Controls → Google Play → approvals for downloads/purchases
3) Manage apps → review and remove risky apps
4) Daily limit / Bedtime → set screen time rules
5) Filters → set age-appropriate content limits

Windows / PC (Microsoft Family Safety)

Microsoft Family Safety
1) Create/add child account
2) Turn on Web & Search Filters
3) Set Screen Time limits
4) Set App & Game Limits if available
5) Review activity weekly

If you need help fast

If something feels off, don’t panic. Screenshot evidence, block, and report.
