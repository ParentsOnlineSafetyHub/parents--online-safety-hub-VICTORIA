Why Device Controls Matter

Most problems happen because default settings are open. Device-level controls give you the strongest baseline: time limits, content filters, purchase locks, and safer defaults — before you even touch an app.

Choose your device type

Pick the closest match. (We’ll keep expanding these guides.)

Apple (iPhone / iPad) Android (Samsung / Google / etc.) Windows Laptop/PC Chromebook

Apple: Screen Time (iPhone/iPad)

1) Turn on Screen Time
Settings → Screen Time → Turn On → This is My Child’s iPhone/iPad.

2) Set a Screen Time Passcode
Settings → Screen Time → Use Screen Time Passcode (keep this private).

3) Downtime (bedtime lock)
Screen Time → Downtime → set bedtime hours (e.g., 8pm–7am).

4) App Limits
Screen Time → App Limits → set daily limits for Social, Games, Entertainment.

5) Content & Privacy Restrictions
Screen Time → Content & Privacy Restrictions → ON
• iTunes/App Store Purchases → Don’t Allow
• Allowed Apps → turn off what you don’t want
• Content Restrictions → age-appropriate

6) Communication Limits (recommended)
Screen Time → Communication Limits → contacts-only where possible.

Android: Family Link (recommended)

1) Install Google Family Link
Parent phone: install “Family Link”
Child phone: follow setup prompts.

2) Screen time & bedtime
Family Link → Screen time → Daily limit + Bedtime.

3) App approvals
Family Link → Controls → Google Play → Require approval for downloads/purchases.

4) Content restrictions
Family Link → Controls → Content restrictions → set age ratings + block explicit content.

5) Location (optional)
Family Link → Location → on (if appropriate for your family).

Windows: Microsoft Family Safety

1) Add the child account
Settings → Accounts → Family → Add a family member.

2) Screen time
Use Microsoft Family Safety to set device/app limits.

3) Content filters
Enable web filters and block adult content.

4) Purchases
Require approval for purchases and disable saved payment methods.

Chromebook: Family Link + Google controls

1) Manage with Family Link
Chromebooks can be supervised using Family Link (child account required).

2) SafeSearch & content filters
Turn on SafeSearch, restrict sites, and control extensions.

3) Time limits
Set daily limits + bedtime lock.
