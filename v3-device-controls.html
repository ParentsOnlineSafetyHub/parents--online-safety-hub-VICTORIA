Device Controls — Lock the device first. Keep it simple.

Why Device Controls Matter

Most problems happen because default settings are open. Device-level controls give you the strongest baseline: time limits, content filters, purchase approvals and safer defaults — before you even touch an app.

Choose your device type

Pick the closest match for your household.

Apple (iPhone / iPad) Android (Samsung / Google / etc.) Windows Laptop / PC Chromebook

Apple: Screen Time

1) Turn on Screen Time
Settings → Screen Time → Turn On → This is My Child’s Device

2) Set a Passcode
Screen Time → Use Screen Time Passcode (keep private)

3) Downtime
Screen Time → Downtime → Set bedtime lock hours

4) App Limits
Screen Time → App Limits → Set daily limits

5) Content & Privacy
Screen Time → Content & Privacy Restrictions → ON

Android: Google Family Link

1) Install Family Link

2) Set Screen Time + Bedtime

3) Require App Approval

4) Enable Content Restrictions

Windows: Microsoft Family Safety

Add child account → Enable screen time → Turn on web filters → Require purchase approval

Chromebook

Supervise using Family Link → Enable SafeSearch → Set time limits
