Priority risks

Twitch is live. Live chat is fast. Strangers can DM, invite, manipulate, or push private contact.

Step-by-step settings

1) Block DMs from strangers
Twitch → Settings → Security and Privacy → Privacy.
Limit whispers/DMs to Friends (or off if possible).

2) Disable chat interaction
If the child is only watching, keep chat closed/disabled where possible.

3) Use device-level controls
Use Screen Time / Family Link to limit access times + restrict mature content.

4) Teach “no off-platform contact”
No Discord invites. No “add me” requests. No DMs.
