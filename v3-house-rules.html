House Rules (One Standard)

Same boundaries. Safer families. Calm rules that reduce arguments.

How to use these rules

These rules work best when your settings match your rules. If you say “no DMs” but DMs are enabled, kids will find the gap. Lock first. Then teach.

Age Guide Device Controls Platform Controls

Non-negotiables (all ages)

1) No private chats with strangers.

2) No secrets with adults online. If someone says “don’t tell your parents” → tell a parent immediately.

3) Location sharing OFF. No Snap Map / live location / “check-ins”.

4) Real-life friends only. Friends list is not a popularity contest.

5) No meeting online contacts. Not alone. Not ever.

6) Devices charge overnight in a shared area. Bedrooms are for sleep.

Account rules (simple + powerful)

Parent controls installs. Kids don’t install new apps without approval.

Private profiles by default. Public profiles are earned, not given.

Password safety. Parents keep recovery access for younger kids. No sharing passwords with friends.

One identity rule. No fake ages, no fake names used to bypass safety.

Communication rules (chat / voice / video)

DMs: friends only. If the app can’t lock DMs, the app doesn’t get used.

Voice chat: Off for younger kids. Friends-only for older kids (earned).

Video calls: Shared spaces for younger kids. No private calls with unknown contacts.

Group chats: Parents can see the group name + members. No “hidden groups”.

Content rules

Age ratings matter. If it’s rated above their age, it’s blocked.

Short-form feeds need limits. TikTok/Shorts/Reels = time windows.

No sexual content. If it appears: exit → block → report → tell parent.

Check-ins (how to supervise without a war)

Weekly 5-minute check (calm tone):
• Privacy settings (still private?)
• Friend list (anyone unknown?)
• Message requests (any strangers?)
• Block list (use it more, not less)

Monthly reset: update passwords, recheck device controls, review new apps/games.

Consequences (calm, consistent, effective)

We don’t explode. We adjust access. Consequences should be predictable:

Level 1: Rule broken → time reduced + settings tightened.

Level 2: Repeat → remove chat/DMs for a period + supervised use only.

Level 3: Serious risk / secrecy / adult contact → pause the app/platform and report.

Key rule: the goal is safety and trust, not punishment.

Red Flags Reporting

Age quick rules (shortcut)

0–6: no social apps, no online chat, shared spaces only.

7–9: friends-only play, no voice chat, parents approve everything.

10–12: no public profiles, no random adds, devices charge in shared area.

13–15: private accounts, DMs friends-only, weekly 5-minute check.

16–17: teach judgment, boundaries, reporting, and “no meeting online contacts alone”.
